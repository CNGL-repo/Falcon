Globic Log Service - Content Processing Activities List and Allowed Content

The following is a list of the content processing activities as stated in the Globic semantic model. Also included in the list are the types of content that the activities can consume and generate.
Note that the Globic model will change over time and this list may become outdated.

This list is accurate as of 11/09/2013.

  1. TextAnalysis
    • Allowable consumed content types:
      • gic:Content
      • nif:String
      • nif:Structure
      • nif:Word
      • nif:Phrase
      • nif:Sentence
    • Allowable generated content types:
      • gic:Content
      • nif:String
      • nif:Word
      • nif:Phrase
      • sso:Phrase
      • sso:Word
  2. Extraction
    • Allowable consumed content types:
      • gic:Content
    • Allowable generated content types:
      • gic:Content
      • gic:SourceUnit
  3. Segmentation
    • Allowable consumed content types:
      • gic:Content
      • gic:SourceUnit
    • Allowable generated content types:
      • gic:Content
      • gic:Segment
  4. TerminologyIdentification
    • Allowable consumed content types:
      • gic:Content
      • nif:String
      • nif:Structure
      • nif:Word
      • nif:Phrase
      • nif:Sentence
      • sso:Phrase
      • sso:Word
    • Allowable generated content types:
      • gic:Content
      • gic:Term
  5. Translate
    • Allowable consumed content types:

      • sso:Word

      • gic:Content
      • gic:Translation
  6. ReviseTranslation

      • gic:Content

      • nif:Structure

      • nif:Phrase
      • sso:Phrase
      • sso:Word

      • gic:TranslationRevision
    • Allowable generated content types:
      • gic:Content
      • gic:TranslationRevision
